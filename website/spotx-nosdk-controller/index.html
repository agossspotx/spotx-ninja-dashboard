spotx-nosdk-controller
README

SpotX's nosdk-controller is the js runtime used by SDK4 (iOS and Android).

This javascript SDK controller lives on the CDN and can be updated at any time, making it easy to fix or add publisher features without requiring publishers to update their client SDK binary.
