spotx-web-nosdk
README

SpotX's SDK settings endpoint.

The purpose of web-nosdk is to serve a JSON blob that contains settings for SDK4. If passed a valid apikey, the settings endpoint returns custom settings corresponding to the apikey. If no apikey is passed, or the passed key is invalid, the settings endpoint returns a set of default settings.
